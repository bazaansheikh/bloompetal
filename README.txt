This project is based on Bloom & Petal Florist, a local flower shop concept. 
I used soft pastel pink colors to create a calm and romantic atmosphere, 
which matches the florist theme.

The layout is built using Flexbox for responsive design. I used Google Fonts 
(Poppins) for modern typography and applied hover effects for interactivity.

The biggest challenge was making the website responsive, which I solved using 
media queries. I also ensured semantic HTML structure for better accessibility.

Overall, the design focuses on simplicity, elegance, and user experience.
